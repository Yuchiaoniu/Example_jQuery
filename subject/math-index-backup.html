inversely-proportional

Inverse proportion, also known as inverse variation, is a relationship between two variables in which their product remains constant.
When two variables are inversely proportional, it means that as one variable increases, the other variable decreases.

Intercept means the point(s) at which a line or curve intersects an axis.

simultaneous-equation

Simultaneous equations, also known as a system of equations, means a set of equations with multiple variables that are solved together.
In a system of simultaneous equations, the goal is to find the values of the variables that satisfy all the given equations simultaneously.

square-root

The square root of a number is a value that, when multiplied by itself, gives the original number.
In other words, it is the value that, when squared, equals the given number.

basic-algebra-formulas

Basic algebra formulas means fundamental equations and rules used in algebraic calculations.

quadratic-equation

A quadratic equation is a second-degree polynomial equation in a single variable,
typically written in the form: ax^2 + bx + c = 0

completing-the-square

Completing the square is a technique used to manipulate a quadratic equation in order to express it in a perfect square form.

parabola

A parabola is a U-shaped curve that is symmetric and can be formed by graphing a quadratic function.

geometry

Geometry is a branch of mathematics that deals with the study of shapes, sizes, properties, and relationships of objects in space.
